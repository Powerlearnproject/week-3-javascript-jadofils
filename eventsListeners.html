Event Listeners and Handlers

Types of Event Listeners

Event listeners are added using the addEventListener method and can be applied to a variety of events. Here are some common types:

Mouse Events

  • click: Fired when an element is clicked.
  • dblclick: Fired when an element is double-clicked.
  • mouseover: Fired when the mouse pointer is moved onto an element.
  • mouseout: Fired when the mouse pointer is moved out of an element.
  • mousemove: Fired when the mouse pointer is moved within an element.

Keyboard Events

  • keydown: Fired when a key is pressed down.
  • keyup: Fired when a key is released.
  • keypress: Fired when a key is pressed down and released (deprecated).

Form Events

  • submit: Fired when a form is submitted.
  • focus: Fired when an element gains focus.
  • blur: Fired when an element loses focus.
  • change: Fired when the value of an element changes.

Window Events

  • load: Fired when the whole page has loaded.
  • resize: Fired when the window is resized.
  • scroll: Fired when the document view is scrolled.

Touch Events

  • touchstart: Fired when a touch point is placed on the touch surface.
  • touchmove: Fired when a touch point is moved along the touch surface.
  • touchend: Fired when a touch point is removed from the touch surface.

Custom Events

  • Custom events can be created and listened to using the CustomEvent constructor.

Types of Event Handlers

Event handlers are assigned directly to an element's event properties. Here are some common types:

Mouse Event Handlers

  • onclick: Handles the click event.
  • ondblclick: Handles the double-click event.
  • onmouseover: Handles the mouseover event.
  • onmouseout: Handles the mouseout event.
  • onmousemove: Handles the mousemove event.

Keyboard Event Handlers

  • onkeydown: Handles the keydown event.
  • onkeyup: Handles the keyup event.
  • onkeypress: Handles the keypress event (deprecated).

Form Event Handlers

  • onsubmit: Handles the submit event.
  • onfocus: Handles the focus event.
  • onblur: Handles the blur event.
  • onchange: Handles the change event.

Window Event Handlers

  • onload: Handles the load event.
  • onresize: Handles the resize event.
  • onscroll: Handles the scroll event.

Touch Event Handlers

  • ontouchstart: Handles the touchstart event.
  • ontouchmove: Handles the touchmove event.
  • ontouchend: Handles the touchend event.
